Project

homebridge-tank-utility

I built this plugin to integrate Tank Utility propane monitors with HomeKit through Homebridge. It exposed sensor data in a way that made tank levels accessible in existing Apple Home dashboards and automations.

The project focused on practical API integration, plugin architecture, and shipping a useful tool for a niche but active community.
