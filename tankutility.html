homebridge-tank-utility

homebridge-tank-utility is a plugin for HomeBridge, the popular NodeJS server which ties unsuported devices into Apple's HomeKit protocol. This plugin integrates Tank Utility devices into HomeKit. The propane tank monitors are exposed to HomeKit as humidity sensors, in order to get a percentage read out. Ideally as Apple expands the HomeKit Accessory Protocol specification, native support for tank levels will be added.

For some time, I've wanted to dive into creating a HomeBridge plugin. I had previously been using HomeBridge to integrate my personal smart devices with pre-existing plugins. When I purchased a TankUtility monitor and discovered that a plugin did not yet exist, it was time for me to dip my toes into plugin development world.

homebridge-tank-utility can be found on both GitHub and the npm website .
